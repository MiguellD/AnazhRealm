## §3 · DIE AUFSTIEGS-LEITER — Selbst-Erweiterung in voller Tiefe (M9)

**Die Vision-Wahrheit:** „DSL-Daten, nie Code" ist die MEMBRAN für Ungeprüftes — nicht die
Decke des Wachstums. Drei Wachstums-Organe existieren schon: fremder VOLL-CODE in Portal-
Welten (Lokalität) · Geste→Gesetz (Laufzeit-Evolution, fitness-selektiert) · der STAMM via
Co-Creation Mensch+KI (dieser Dialog; die ~3600 Invarianten sind das Immunsystem für
Code-Mutationen). Die Leiter verbindet sie zu EINEM Aufstieg — jede Sprosse hat WAND + TOR:

1. **REICHEN** (✓ V18.152): die Welt reicht DSL-Fähigkeiten → Quarantäne (3 Wände).
2. **GEWÄHREN** (✓): souveräne R2-Geste; R4-Herkunft reist, Rückruf wirkt immer.
3. **BEWÄHREN** (NEU): gewährte Fähigkeiten bekommen den GESETZ-Lebenszyklus — dieselbe
   `_worldRuleFitness`-Wertung (Vorhersagefehler-δ: hat der Lauf die Welt verbessert?);
   Läufe zählen, Fitness reift SICHTBAR (die Karte zeigt sie — der Spieler SIEHT, was
   sich bewährt). Kompositions-Tiefe: Fähigkeiten dürfen Fähigkeiten rufen (DSL-Refs,
   Budget transitiv — das Fraktal-Bauplan-Muster auf Programme).
4. **KRISTALLISIEREN** (NEU): eine bewährte Fähigkeit (fitness ≥ Schwelle, ≥N Läufe,
   wand-sicher) wird durch DEINE Geste VOKABULAR — ein benannter Chat-/DSL-Eintrag der
   Welt (das E4-Kristall-Muster eine Ebene höher). Sie reist signiert + mit Kette im
   Manifest — andere Welten können sie GEREICHT bekommen (die Fähigkeit WANDERT durchs
   Ultiversum: Welt A gebiert, du bewährst, Welt B empfängt — Mesh-Kultur).
5. **DER CODE-PFAD** (die letzte Sprosse, der ehrliche Gigant): eine Welt darf CODE
   vorschlagen (ein Modul, einen Shader, eine Welt-Mechanik) — er betritt NIE die
   Laufzeit-Autorität. Sein Weg ist der CO-CREATION-KANAL: Vorschlag → die KI (ich)
   liest/prüft/destilliert ihn im UNEXPONIERTEN Pfad (Dual-LLM: der Vorschlag informiert,
   formuliert aber nie) → du adoptierst souverän → die Invarianten gaten → der Stamm
   wächst um einen Ring. Das ist KEINE Schwäche („nur mit Mensch") — es ist die Lösung
   des Problems, an dem die Profis 2026 real scheitern (ungeprüfte Selbst-Modifikation =
   lethal trifecta; Darwin-Gödel validiert genau diese Form: empirisch testen + der

   FÜTTERT die nächste, und nichts Bewährtes geht verloren (Fitness + Kette + Archiv).

**Warum das der Gigant der Neuzeit ist:** offene Mutation (Welten = Voll-Code) ×
